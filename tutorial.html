Tutorials

The examples below are available as Jupyter notebooks in the GitHub repository.

Examples of using PyCCE

  • NV Center in Diamond
  • VV in SiC
  • Shallow donor in Si
  • Correlation function
  • Multiple central spins
  • Dissipative spin bath

The recommended order of the tutorials is:

  • NV Center in Diamond goes through the Quick Start example in more details.

  • VV in SiC explores the difference between generalized CCE with and without random bath state sampling. Also, in this example we introduce the way to work with hyperfine tensors computed from DFT.

  • Shallow donor in Si shows how to include custom hyperfine couplings for more delocalized defects in semiconductors.

  • Correlation function explains how to use autocorrelation function of the noise to predict the decay of the coherence of the NV center in diamond.

  • Multiple central spins goes over systems with two central spins, either forming a hybrid qubit, or two entangled qubits.

  • Dissipative spin bath provides details on the master-eqation based solvers ME-CCE and ME-gCCE.
